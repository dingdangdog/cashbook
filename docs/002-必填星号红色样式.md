# 002：必填星号红色样式

## 变更时间

2026-03-23

## 问题

修改密码等使用 `UiTextInput` 且带 `required` 的表单，标签后的必填 `*` 与标签同色（`text-foreground/80`），未突出显示为红色。

## 修改

- `app/components/ui/TextInput.vue`：将必填 `*` 放入独立 `<span>`，使用 `text-red-500`，与同组件错误文案颜色一致。

## 影响范围

所有带 `label` + `required` 的 `UiTextInput` 实例（含修改密码对话框、登录/注册等）。
